Qu'est ce qu'une Machine Virtuelle

Bienvenue dans cette présentation sur la virtualisation.

Avant de comprendre la machine virtuelle, il faut définir ce qu'est un ordinateur : c'est un appareil électronique capable de traiter des données et d'exécuter des tâches grâce à deux composants essentiels :

  • Le Hardware : La partie physique (CPU, RAM, Disque dur).
  • Le Software : Les programmes et le système d'exploitation qui contrôle le matériel.

Qu'est-ce qu'une VM ?

Une Machine Virtuelle (VM) est un ordinateur logiciel qui fonctionne à l'intérieur d'un ordinateur physique. C'est un "ordinateur dans l'ordinateur" qui possède son propre système d'exploitation (comme Ubuntu), sa mémoire et son stockage.

Le rôle de l'Hyperviseur

Les machines virtuelles sont créées par un logiciel appelé Hyperviseur. Son rôle est de partager les ressources du processeur et de la RAM de l'ordinateur réel. Des logiciels comme VirtualBox ou VMware sont des hyperviseurs.

Pourquoi l'utiliser dans notre projet ?

Les VM sont très utiles pour exécuter différents systèmes d'exploitation sur une seule machine et tester des logiciels en toute sécurité.

Les Avantages :

  • Flexibilité : Gain de temps et économie de matériel physique.
  • Sécurité : Chaque VM est isolée ; une erreur dans la VM n'affecte pas votre PC.

Les Inconvénients :

  • Ressources : Elles consomment beaucoup de RAM et d'espace disque.
  • Performance : Elles peuvent être plus lentes qu'un ordinateur réel.

Notre avis sur cette technologie

Selon nous, la machine virtuelle est un outil simple et puissant. Pour notre SAÉ, cela a été indispensable : cela nous a permis de simuler un environnement serveur professionnel sur nos ordinateurs personnels sans prendre aucun risque technique.
