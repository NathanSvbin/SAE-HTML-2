Conception du site

La création de ce site a été une étape clé pour documenter notre travail sur la SAÉ S1.03.

Design et Identité Visuelle

Pour garantir une cohérence visuelle, nous avons défini une palette de couleurs moderne. Le design repose sur un conteneur central épuré pour faciliter la lecture des rapports techniques.

Technologies utilisées

Le site a été développé avec des technologies standards du web pour assurer légèreté et compatibilité :

  • HTML5 : Pour structurer le contenu de nos différentes pages.
  • CSS3 : Pour la mise en page personnalisée.
  • Bootstrap 5 : Utilisé principalement pour la barre de navigation, le pied de page et les différents container.

Flux de travail et Hébergement

Le développement ne s'est pas fait directement sur le serveur. Nous avons mis en place un flux de travail professionnel :

  1. Développement local : Rédaction du code sur nos machines personnelles.
  2. Transfert SFTP : Utilisation de FileZilla pour envoyer les fichiers vers notre machine virtuelle Ubuntu de manière sécurisée.
  3. Déploiement : Les fichiers sont stockés dans le répertoire /var/www/html/ de notre serveur Apache2, ce qui les rend accessibles via le réseau.
